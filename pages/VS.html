¿Qué es?

Un Servidor Virtual o un VPS (Servidor Privado Virtual) es una réplica o representación basada en software de un servidor físico. Se utiliza comúnmente en el servicio de alojamiento web VPS, almacenando todos los datos y archivos de su sitio web en un servidor virtual privado. El alojamiento VPS es un intermediario perfecto entre una solución de alojamiento web compartida y dedicada.
